Text-Annotated GUI

Machine Learning document annotations for the average Joe.

  • Start Annotating

Keywords

Searching for specific words or phrases can be a tedious process for large files. Ctrl+F is a simple solution for when you know the exact word that you want, but what happens when the problem is more complex than that? How can you highlight words that belong to an umbrella category? TAG aims to solve this problem.

Machine Learning

By specifying a category or 'label', a user can expect to have their documents highlighted with all words or phrases that match that label. This is accomplished by sending a sample set to a powerful back-end engine that will intelligently parse the document for any occurrences of a certain category.

Annotations

Once the machine has completed its computations, it will return a list of its predictions from the documents. These results are then displayed graphically on the web interface as annotated highlights for the user to digest.

Features

Intelligent Highlights

Annotate a document by defining a label and submitting it to the back-end machine for highlights!

Data Retention

Retain the state of your highlights by creating an account - you can reload your progress next time you login.

Configuration

We offer the ability to truly customize the application to your needs. You can upload multiple files, select your highlight colors, define your labels, and more!

Platforms

In addition to our web page, we also offer this project as a standalone application for iOS on the App Store.

Zip Files

Have a bulk amount of files to annotate? We support .zip!

Grade A Performance

This application works like none other. The best on the market, some say. And it's free!
